Throwing ball to target 3D game.
Written in python 3.6 with VPython.
Run file Program.py

Instructions:
Scroll - zooming
Right Mouse Click - changing perspective
Left Mouse Click - throwing ball, holding increase power
Mouse Movement - moving ball on the floor edge

Game have adjustment parameters in params.json file. Parameters meaning:
ringThickness - ring border thickness
targetSpeed - ring speed
ballRadius - radius of the ball
ringRadius - radius of the ring
ringHeight - how high from the floor is the ring center
floorRadius - half of floor edge size
wallThickness - this case just floor thickness
g - gravity acceleration
increaseStep - how fast the force grows when holding left mouse button
